Plugin Demo: Lyra Test

Demo details

This is a variant of the Echo Test demo that uses lyra-js, a WebAssembly port of Lyra, to use that codec as the audio stream in the PeerConnection. It is partly based on the excellent lyra-demo project, in particular in terms of how raw audio frames are encoded in JavaScript (and fed to the MediaStream via Insertable Streams), and decoded accordingly on the way in. Considering Insertable Streams are used, the code of the demo will be similar to e2etest.html/.js. Besides, since only audio is relevant, unlike the regular Echo Test demo video and data channels are disabled in this context.

Press the Start button above to launch the demo.
